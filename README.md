# Hero Battle Royale

A complete battle royale game featuring original characters inspired by anime superhero aesthetics.

## Features

- **2 Playable Characters**: Katero (aggressive speedster) and Mizuno (support/control)
- **Directional Melee Combat**: Direction-based attacks with varying damage
- **Unique Ability Systems**: Alpha, Beta, Gamma, and Special abilities per character
- **Team-Based Multiplayer**: 2 teams competing in arena combat
- **Real-time Combat**: Health management, cooldowns, stat buffs
- **Win Conditions**: Team with most eliminations or last team standing

## Characters

### 一人で (Katero Mikitu)
**Savior from the Shadows** - Aggressive speedster with self-sustain
- **Basic Attack**: Infinite combo (20 dmg per hit, 65 final)
- **Alpha**: Speed Blitz (100 dmg, reaches any opponent)
- **Beta**: Triple Strike (100/100/250 dmg)
- **Gamma**: Dark Aura (60% speed, 30% damage, regen 20 HP/5s)
- **Special**: Revival/Heal (2 min cooldown)

### 水のかの (Mizuno Kanade)
**Tide Bringer** - Graceful support with crowd control
- **Basic Attack**: Flowing combo (18 dmg per hit, 60 final)
- **Alpha**: Water Sphere (80 dmg, knockback)
- **Beta**: Tidal Wave (120 dmg, stun 1.5s)
- **Gamma**: Aqua Shield (40% damage reduction)
- **Special**: Healing Rain (heal allies, 90s cooldown)

## Controls

- **WASD**: Movement
- **Mouse**: Look/Target
- **LMB**: Basic Attack
- **Q**: Alpha Ability
- **E**: Beta Ability
- **R**: Gamma Ability
- **Space**: Special Ability
- **ESC**: Pause

## Build

1. Clone repository
2. Open in Unity 2022 LTS+
3. Load Main scene
4. Build for Windows
